v1.3.1

重要提示：
同時進行多項動作的功能仍然可以使用但是預設關閉。如果想要開啟只需點擊「設定」 -> 「進階設定」，設定「開始新動作時」為「無」即可。
新特性：
- 軟體已升級為通用軟體。iPad 上面的展示比以往更好。
- 「提醒我」功能會每若干分鐘(具體可設定)彈出提示問您在做什麼。如果您經常忘記切換動作的話，這是個非常好用的功能。該功能可在「設定」 -> 「進階設定」中開啟。
- 「批次刪除」功能。該功能在刪除大量的歷史記錄時更加方便。

提升和改進：
- 如有需要的話 CSV 報告中的日期可包含年份 (參見「設定」->「報告」) 
- CSV 檔案中的持續時間格式更加精確
- 改進備份檔案的命名 
- 備份清單中的備份檔案可以透過簡單的划動手勢來進行刪除
- 大幅改進圖示和字型在非視網膜螢幕下的顯示效果

漏洞的修補：
- 修正同時進行多項動作時錯誤的倒數計時顯示

v1.2

感謝大家積極的建議和意見反應！

這個新版本花了很多的時間來準備，最後這次的更新包含許多新功能和功能的提昇。下面是「新功能」 的一覽表。您可以透過拜訪我們在 www.aloggers.com 的應用程式 Blog，得到更多詳細資訊並學習如何使用最新的功能。

另外，請不要忘記關注我們的 Facebook 頁面和 Twitter。

新功能：
★ 「目標」功能
★ 按活動類型分組的歷史清單
★ 支援 Twitter 發文
★ 封存活動類型
★ 2 個新的佈景主題

改進：
✓ 在歷史標籤內的日/週/月選擇按鈕上新增「今天」按鈕
✓ CSV 報告中的活動持續時間精準到小數位數
✓ 加入「最小活動持續時間」功能 => 活動持續時間小於最小值時不會被儲存
✓ 在報告中加入百分及總計時間
✓ 在選擇多個活動類型時，加入「全選/取消全選」按鈕，
✓ 加入「移動到」按鈕方便群組功能 (資料夾)
✓ 加入瑞典文翻譯

修復的錯誤：
✝ 持續時間的錯誤
✝ 在日曆檢視中上午下午的時間顯示 (AM/PM)
✝ 非英語語系的 HTML 報告問題
